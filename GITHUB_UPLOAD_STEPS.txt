GITHUB UPLOAD STEPS

Use these steps after creating an empty repository on GitHub.

1. Open the new repository page on GitHub.

2. Copy the repository URL.
   Example:
   https://github.com/your-username/jarvis.git

3. Open PowerShell on the PC that has the project.

4. Run these commands:

   cd D:\Jarvis

   git config user.name "Your Name"
   git config user.email "you@example.com"

   git add .
   git commit -m "Initial J.A.R.V.I.S release"

   git branch -M main
   git remote add origin https://github.com/your-username/jarvis.git
   git push -u origin main

5. Refresh the GitHub page.

6. The repo should now show the uploaded project files.

IMPORTANT

- These steps assume the GitHub repository is empty.
- If a README, license, or other file was already created on GitHub, the remote
  is not empty and the push flow changes slightly.
- If that happens, stop there and use a pull-first flow instead.
